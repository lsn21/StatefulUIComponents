# StatefulUIComponents

[![Version](https://img.shields.io/cocoapods/v/StatefulUIComponents.svg?style=flat)](https://cocoapods.org/pods/StatefulUIComponents)
[![License](https://img.shields.io/cocoapods/l/StatefulUIComponents.svg?style=flat)](https://cocoapods.org/pods/StatefulUIComponents)
[![Platform](https://img.shields.io/cocoapods/p/StatefulUIComponents.svg?style=flat)](https://cocoapods.org/pods/StatefulUIComponents)

A collection of customizable `@IBDesignable` UI components with state-specific properties.

## Components

### StatefulUIButton
A UIButton subclass with state-specific properties for background color, title color, font, and number of lines.

### StatefulUITextField (coming soon)
A UITextField subclass with state-specific properties.

## Installation

### CocoaPods

Add the following to your Podfile:

```ruby
pod 'StatefulUIComponents'
